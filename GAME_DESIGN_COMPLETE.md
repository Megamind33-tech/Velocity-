## 1. CORE GAMEPLAY SYSTEMS

### 1.1 PROGRESSION LEVELS (5 Total)
```
LEVEL 1: NOVICE (Mode A - Waypoint Flying)
├─ 5 Simple Songs (C Major, basic notes, 4-8 note phrases)
├─ Hit Zones: ±3 semitones (very forgiving)
├─ Duration: 60-90 seconds per song
├─ Unlock Requirement: Beat ANY 3 songs with 1+ star
└─ Mode: Individual waypoint hitting

LEVEL 2: INTERMEDIATE (Mode A - Waypoint Flying)  
├─ 8 Melodic Songs (some chromatic notes, 6-12 note phrases)
├─ Hit Zones: ±2 semitones (moderate)
├─ Duration: 90-120 seconds
├─ Unlock Requirement: Beat ANY 3 songs with 1+ star
└─ Mode: Individual waypoint hitting with faster tempo

LEVEL 3: ADVANCED (Mode A - Waypoint Flying)
├─ 10 Complex Songs (chromatic, wide range, 8-16 note phrases)
├─ Hit Zones: ±1 semitone (tight)
├─ Duration: 120-180 seconds
├─ Unlock Requirement: Beat ANY 3 songs with 1+ star
└─ Mode: Rapid transitions between notes

LEVEL 4: MASTER (Mode C - Melodic Contour Flying)
├─ 8 Advanced Songs (requires smooth control, legato, vibrato)
├─ Hit Zones: ±1 semitone continuous tracking
├─ Duration: 180-240 seconds
├─ Unlock Requirement: Beat ALL Level 3 songs (10 songs required)
├─ Mode: Smooth melodic curve following
└─ Vibrato Training: Allowed in specific zones

LEVEL 5: LEGEND (Mode C - Melodic Contour Flying)
├─ 8 Professional Songs (extreme difficulty, complex rhythms)
├─ Hit Zones: ±0.5 semitone (professional standard)
├─ Duration: 240-300 seconds
├─ Unlock Requirement: Beat ALL Level 4 songs (8 songs required)
├─ Mode: Professional vocal control
└─ Full Expression: Vibrato, dynamics, legato required
```

**Total Songs in Game: 39 songs** (provides 40+ hours of content)

## 2. STAR RATING SYSTEM (Per Song)

```
⭐ 1 Star: Hit 60-74% of notes correctly
⭐⭐ 2 Stars: Hit 75-89% of notes correctly  
⭐⭐⭐ 3 Stars: Hit 90%+ of notes correctly + smooth transitions

SCORING MULTIPLIERS:
- Combo (5+ correct in a row): 1.25x
- Accuracy (90%+): 1.5x
- Smooth (no jerky transitions): 1.2x
- Perfect (note-perfect throughout): 2x

MAX SCORE: 1000 points per song
```

## 4. REALISTIC PLANE VISUALS

### 4.1 PLANE DESIGN
```
TECHNICAL SPECIFICATIONS:
- Plane Model: Cessna-172 style (single engine, high wing)
- Size on Screen: ~60-80px width
- Visible Elements:
  * Fuselage (body) - white/silver
  * Wings - visible from top angle
  * Tail - visible rudder/elevator
  * Cockpit window - small highlight
  * Status Lights - green (good), yellow (caution), red (warning)

PLANE ORIENTATION:
- Facing right (direction of flight)
- Pitch indicator (nose up/down shows altitude)
- Bank indicator (slight tilt for smooth transitions)
- Glow effect when hitting notes correctly
```

### 4.2 FLIGHT MECHANICS
```
ALTITUDE SYSTEM:
- Y position = vocal pitch
- Smooth interpolation (0.2s smoothing)
- Visual altitude readout (Hz display)
- Reference altitude lines (every semitone)

SPEED SYSTEM:
- X position = song progression
- Constant forward speed
- Visual speed indicator
- Time remaining display

HEADING/DIRECTION:
- Always flying right
- Bank angle shows pitch stability
- Compass heading: 090° (due east) constant

STATUS INDICATORS (HUD):
- Altitude: [note name] [octave] [Hz]
- Speed: [song progress %]
- Vertical Speed: [pitch change rate]
- G-Force: [smoothness rating]
- Flight Time: [elapsed / total]
```

### 4.3 VISUAL FEEDBACK
```
CORRECT HIT:
- Plane flashes white/cyan
- Glow pulse around plane
- Trail effect (colorful path)
- +10 points HUD notification

NEAR MISS:
- Plane flashes yellow
- Warning beep
- Turbulence effect (slight wobble)
- Accuracy indicator shows "CLOSE"

MISS:
- Plane flashes red
- Error buzzer
- Shaking effect
- -25 points HUD notification
- "MISSED" popup

SMOOTH TRANSITION:
- Plane has smooth trail
- Blue/cyan glow indicates smoothness bonus
- "SMOOTH" bonus badge
- +5 bonus points
```

## 5. WAYPOINT REVEAL SYSTEM (Flappy Bird Style)

### 5.1 GRADUAL REVEAL
```
NOVICE LEVEL:
- Waypoints visible 3 seconds ahead
- Large circles (easy to target)
- Clear note names displayed
- Reference lines showing octaves

INTERMEDIATE LEVEL:
- Waypoints visible 2.5 seconds ahead
- Medium circles
- Note names with pitch indicators
- Grid lines for reference

ADVANCED LEVEL:
- Waypoints visible 2 seconds ahead
- Smaller circles
- Just note names (no Hz display)
- Minimal reference lines

MASTER LEVEL:
- Waypoints visible 1.5 seconds ahead (melodic curve shows full path)
- Target line continuously visible
- No waypoint circles (continuous curve)
- Real-time pitch feedback only

LEGEND LEVEL:
- Waypoints visible 1 second ahead (curve shows full path)
- Minimal visual aids
- Must rely on ear training
- Real-time feedback only
```

### 5.2 OBSTACLE/CHALLENGE VARIATION
```
TEMPO VARIATIONS:
- Song duration affects waypoint timing
- Faster songs = denser waypoint spacing
- Slower songs = more time to adjust

INTERVAL VARIATIONS:
- Small intervals (1-2 semitones): Easy
- Medium intervals (3-5 semitones): Moderate
- Large intervals (6+ semitones): Challenge
- Visual difficulty indicator per waypoint

RHYTHM VARIATIONS:
- Steady rhythm (regular spacing): Beginner
- Syncopated rhythm (irregular spacing): Intermediate
- Complex rhythm (unexpected timing): Advanced
```

## 6. GAME LOOP STRUCTURE

### 6.1 SESSION FLOW
```
1. HOME SCREEN
   ├─ Select Level (unlocked levels only)
   └─ Select Song (locked songs grayed out)

2. SONG SELECTION
   ├─ Show song difficulty
   ├─ Show personal best
   ├─ Show target stars for unlock
   └─ Ready to Play

3. PRE-GAME SETUP
   ├─ Choose difficulty/speed (optional)
   ├─ Warm-up vocal range (optional)
   └─ Start Game

4. GAMEPLAY
   ├─ Load plane visuals
   ├─ Display waypoints/melodic curve
   ├─ Real-time pitch tracking
   ├─ Score accumulation
   └─ Pause/Resume available

5. POST-GAME RESULTS
   ├─ Final score calculation
   ├─ Star rating awarded
   ├─ Personal best comparison
   ├─ Achievements unlocked
   ├─ Statistics updated
   ├─ XP earned
   └─ Option to Retry or Go Home

6. UNLOCK CHECK
   ├─ Did player beat 3 songs in current level?
   └─ If yes: Next level unlocks automatically
```

### 6.2 PROGRESSION GUARDRAILS
```
SAFETY SYSTEMS:
- Can always replay any unlocked level
- Can play songs below current level
- No time limits on attempts
- No "game over" (just low scores)
- Unlimited retries (builds persistence)

DIFFICULTY SCALING:
- Early songs introduce mechanics gradually
- Hit zones widen for lower levels
- Waypoint reveal timing generous in early levels
- Vibrato only required at Level 4+
- Speed variations optional in practice mode
```

## 8. XP & PROGRESSION CURRENCY

```
XP EARNINGS (Per Song):
- Base: 50 XP × (stars achieved)
  * 1 star = 50 XP
  * 2 stars = 100 XP
  * 3 stars = 150 XP

- Accuracy Bonus: +10 XP per 10% accuracy
  * 80% = +80 XP
  * 90% = +90 XP
  * 100% = +100 XP

- Combo Bonus: +1 XP per combo (5+ combo)
  * 10 combo = +10 XP
  * 20 combo = +20 XP
  * Max = 100 XP (50+ combo)

- Smoothness Bonus: +50 XP if "Smooth" achieved
- First-Time Bonus: +100 XP on first play of a song
- Difficulty Bonus: +25 XP × difficulty level (L1=25, L5=125)
- Daily Multiplier: 1.1x - 2x (based on daily streak)

MAX PER SONG: 500-600 XP (with bonuses)

TOTAL XP TRACKING:
- Level 1 Completion: 1000-1200 XP
- Level 2 Completion: 2000-2400 XP
- Level 3 Completion: 3000-3600 XP
- Level 4 Completion: 2500-3000 XP (harder songs, fewer)
- Level 5 Completion: 2500-3000 XP (ultimate challenge)

CUMULATIVE TOTALS:
- Beat all 39 songs: 15,000-18,000 XP
- Get 3 stars on all 39 songs: 25,000-30,000 XP (lifetime goal)
```
